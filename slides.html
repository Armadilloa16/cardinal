Contents

Cardinal is a package for R designed to be used for the analsis of mass spectrometry imaging data, I'll demonstrate:

  • Reading in IMZML data.
  • Plotting Spectra.
  • Plotting Masses.
  • Peakpicking.
  • PCA.

Reading Data

Reading IMZML data is trivial. Cardinal can also read Analyze 7.5 format imaging data if needed.

d.name = "example"
d.fold = "data"
data   = readImzML(d.name, d.fold)

All the relevant information about the data can be obtained through the functions mz, coord, spectra, features, pixels, and some overwritten base functions such as dims, nrow, and ncol, for example:

dim(data)
## Features   Pixels 
##     1001    51783

Plotting Individual Spectra

plot(data, coord = list(x = 678, y = 209))

Plotting Mean Spectra

plot(data, coord = list(x = 678, y = 209), plusminus = 5)

Plotting Multiple Spectra

plot(data, pixel=spec, pixel.groups = group, 
     superpose = TRUE, key = TRUE, col = c("blue", "red", "magenta"))

Plotting Particular Masses

image(data, mz = 1298, plusminus=0.3)

Peakpicking and other Preprocessing

Can be performed using various provided functions, including normalisation (normalize), smoothing (smoothSignal), baseline reduction (reduceBaseline), peak picking (peakPick), and peak alignment (peakAlign).

Cardinal also includes some useful functions for other common preprocessing operations such as resamping/ binning (reduceDimension) and filtering rare peaks (peakFilter).

PCA Loadings

pca = PCA(data, ncomp = 1)
plot(pca)

PCA Scores

image(pca)
